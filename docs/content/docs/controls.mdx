# Controls

| Key | Action |
| --- | --- |
| `Enter` | Select, confirm, or start download |
| `f` | Toggle a channel as favorite |
| `PageUp` / `PageDown` | Move through search results |
| `Escape` | Go back |
| `q` | Quit safely |

All screens are designed to be used entirely with the keyboard.

Tip: Press `Ctrl + K` (or `Cmd + K`) in the docs site to open search.
