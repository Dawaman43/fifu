# Features

Fifu is optimized for fast, repeatable YouTube downloads without leaving the terminal.

## Core capabilities

- Channel search with popularity sorting
- Direct playlist downloads by URL
- Quality presets for best, 1080p, 720p, 480p, and audio-only
- Subtitle download and embedding
- Concurrent downloads with live progress
- Automatic duplicate skipping based on title
- Favorites and recent history for quick re-entry

## What makes it fast

- Metadata fetch runs asynchronously
- Downloads run in parallel (up to three at a time)
- Already-downloaded titles are skipped before queuing

## What makes it safe

- Downloads are stored in per-channel folders
- Logs are written locally for debugging
- Favorites and history live in a small JSON file you control
