# Glossary

## Channel

A YouTube creator profile that hosts uploads and playlists.

## Playlist

A curated list of videos. You can paste a playlist URL to download only that list.

## Queue

The list of videos that Fifu will download in the current session.

## Quality preset

A predefined `yt-dlp` format string that selects best, 1080p, 720p, 480p, or audio-only streams.

## Subtitles

Text tracks that can be downloaded and embedded into the final file.

## History

The last 10 searches you performed, stored locally in `~/.config/fifu/data.json`.

## Favorites

Channels you marked with the `f` key for quick access.
