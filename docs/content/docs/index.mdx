# Fifu documentation

Fifu (Fetch It For Us) is a fast, keyboard-first TUI for downloading YouTube channel videos and playlists. Built with Textual and powered by `yt-dlp` for reliable extraction and clean progress reporting.

## A quick mental model

1. **Launch**: Open your terminal and run `fifu`.
2. **Search**: Find a channel or paste a direct URL.
3. **Configure**: Set your quality and subtitle preferences.
4. **Run**: Monitor live progress as videos are downloaded.
